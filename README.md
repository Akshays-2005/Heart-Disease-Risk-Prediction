## ğŸ“˜ Short Project Summary

The Heart Disease Prediction System is a modern, responsive web application that predicts the likelihood of heart disease using 13 clinical parameters. Users enter medical values such as age, cholesterol, ECG results, and more. These inputs are analyzed by a Machine Learning model (Random Forest) which returns a clear risk prediction:

- Low Risk

- Medium Risk

- High Risk

This project is designed for students, healthcare enthusiasts, and researchers, combining clean UI/UX with real-world ML inference.
